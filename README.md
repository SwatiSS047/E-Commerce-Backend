Architected a scalable E-Commerce REST API with three-tier 
role-based access control (Admin, Merchant, Customer) using 
Spring Security and JWT authentication.

Key Features:
- OTP-based email verification using Redis for fast 
  in-memory storage
- Razorpay payment gateway integration for cart, order 
  and payment lifecycle
- Admin module for merchant/customer management and 
  product approval workflows
- Role-based access control using @PreAuthorize annotations

Tech Stack: Java | Spring Boot | Spring Security | JWT | 
Redis | Razorpay | MySQL | Oracle | Git | Postman
